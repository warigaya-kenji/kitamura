# netshop-spa

ECサイトリニューアル フロントエンド（ネットショップ SPA）


### バージョン
# node v14.15.1
# npm  6.14.8
# @vue/cli 4.5.9
# vuetify 2.4.3

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```
or
```
npm run serve-dev
```

### Compiles and minifies for production
```
npm run build-dev
```

### Compiles and minifies for production
# ```
# npm run build-prod
# ```

### Lints and fixes files
```
npm run lint
```
